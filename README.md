# ch4ser-vim
The Neovim configuration file for ctf's web dog

## how to use

### make sure you have Neovim installed
```
apt install neovim
```

### then clone the repo to ~/.config/nvim
```
mkdir .config
cd .config
curl -fLo ~/.config/nvim/ --create-dir https://github.com/ch4ser-go/ch4ser-vim
